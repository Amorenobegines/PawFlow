# 🐾 PawFlow

A Laravel web application for managing a dog grooming salon.

## Features

- Authentication (Login & Register)
- Role management (Admin / Employee)
- Client management
- Pet management
- Service management
- Appointment scheduling
- Full CRUD operations
- MySQL database

## Technologies

- Laravel 12
- PHP 8.3
- MySQL
- Blade
- HTML5
- CSS3
- JavaScript

## Installation

Clone the repository:

```bash
git clone https://github.com/Amorenobegines/PawFlow
```

Install dependencies:

```bash
composer install
```

Install Node dependencies:

```bash
npm install
```

Copy the environment file:

```bash
cp .env.example .env
```

Generate the application key:

```bash
php artisan key:generate
```

Run migrations:

```bash
php artisan migrate
```

Start the server:

```bash
php artisan serve
```
